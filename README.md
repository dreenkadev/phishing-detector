# Phishing Detector

Phishing URL detection tool.

Alat deteksi URL phishing.


## Description

Analyzes URLs and web content to identify phishing attempts using pattern matching and domain analysis.

## Deskripsi

Menganalisis URL dan konten web untuk mengidentifikasi upaya phishing menggunakan pencocokan pola dan analisis domain.


## Installation

- Navigate to the project folder
- No dependencies required

## Instalasi

- Masuk ke folder proyek
- Tidak perlu dependensi


## Usage

- Detect: node phishing_detect.js https://suspicious-site.com
- Demo: node phishing_detect.js --demo

## Penggunaan

- Deteksi: node phishing_detect.js https://suspicious-site.com
- Demo: node phishing_detect.js --demo
